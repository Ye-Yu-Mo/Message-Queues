详细描述

SQLite 数据库操作助手类

此类封装了 SQLite 数据库的基本操作，包括创建/打开数据库、 执行 SQL 语句和关闭数据库的功能。

主要功能：

  • 创建和打开数据库
  • 执行 SQL 语句（包括表操作和数据操作）
  • 关闭数据库
