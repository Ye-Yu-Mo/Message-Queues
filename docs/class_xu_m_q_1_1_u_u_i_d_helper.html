详细描述

提供生成 UUID 的工具类。

该类包含一个静态方法用于生成唯一标识符（UUID） UUID 格式为 8-4-4-4-12 的十六进制字符串

成员函数说明

◆ uuid()

static std::string XuMQ::UUIDHelper::uuid ( )
inlinestatic

生成一个随机 UUID。

该方法使用随机设备和伪随机数生成器生成一个 128 位的随机数 将其格式化为 UUID 字符串。UUID 字符串的格式为 8-4-4-4-12，其中每部分用连字符分隔。

返回
std::string 生成的 UUID 字符串。
