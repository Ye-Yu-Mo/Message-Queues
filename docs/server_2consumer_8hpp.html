消费者管理模块

本文件定义了消费者及其管理类，用于管理队列中的消费者。 提供了消费者的创建、移除、选择等功能，以及对消费者队列的管理。 主要包括以下结构和类：

  • Consumer: 表示单个消费者的结构
  • QueueConsumer: 管理单个队列中的多个消费者
  • ConsumerManager: 管理所有消费者队列
