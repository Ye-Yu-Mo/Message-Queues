工具类封装

此文件定义了 SqliteHelper StrHelper UUIDHelper FileHelper

SqliteHelper 类提供以下功能：

  • 创建和打开 SQLite 数据库
  • 执行 SQL 语句，包括表和数据操作
  • 关闭数据库

StrHelper 类提供字符串处理功能，例如根据指定分隔符分割字符串。

UUIDHelper 类提供生成静态方法用于生成唯一标识符（UUID），格式为 8-4-4-4-12 的十六进制字符串。

FileHelper 类提供以下功能

  • 检查文件是否存在
  • 获取文件大小
  • 读取和写入文件
  • 重命名文件
  • 创建和删除文件及目录
